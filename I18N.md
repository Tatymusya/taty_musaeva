# 🌍 Локализация (i18n)

Система динамической локализации с поддержкой **русского** и **английского** языков.

## 📁 Структура

```
src/core/i18n/
├── types.ts           # Типы для переводов
├── i18n.ts           # Основной класс локализации
├── index.ts          # Экспорты
└── locales/
    ├── ru.ts         # Русские переводы
    └── en.ts         # Английские переводы
```

## 🚀 Использование

### Базовое API

```typescript
import { I18n } from '@core/i18n';

// Получить текущий язык
const locale = I18n.getLocale(); // 'ru' | 'en'

// Установить язык
I18n.setLocale('en');

// Переключить язык
I18n.toggleLocale();

// Получить перевод
const text = I18n.t('nav.home'); // 'Главная' или 'Home'

// Вложенные ключи
const desc = I18n.t('hero.description');
```

### HTML с data-i18n

```html
<!-- Текстовый контент -->
<p data-i18n="hero.greeting">Привет, меня зовут</p>

<!-- Placeholder для input -->
<input data-i18n-placeholder="contact.formName" placeholder="Ваше имя" />

<!-- Кнопки -->
<button data-i18n="contact.formSubmit">Отправить</button>
```

### Подписка на изменения

```typescript
// Подписаться на изменение языка
const unsubscribe = I18n.subscribe(locale => {
  console.log('Язык изменён на:', locale);
  // Обновить UI если нужно
});

// Отписаться
unsubscribe();
```

## 📝 Добавление новых переводов

### 1. Обновите типы (types.ts)

```typescript
export interface Translation {
  // ...
  newSection: {
    title: string;
    description: string;
  };
}
```

### 2. Добавьте переводы в ru.ts

```typescript
export const ru: Translation = {
  // ...
  newSection: {
    title: 'Заголовок',
    description: 'Описание',
  },
};
```

### 3. Добавьте переводы в en.ts

```typescript
export const en: Translation = {
  // ...
  newSection: {
    title: 'Title',
    description: 'Description',
  },
};
```

### 4. Используйте в HTML

```html
<h2 data-i18n="newSection.title">Заголовок</h2>
<p data-i18n="newSection.description">Описание</p>
```

## 🔧 Конфигурация

### locale.ts

```typescript
private config: LocaleConfig = {
  locale: 'ru',              // Язык по умолчанию
  fallbackLocale: 'ru',      // Резервный язык
  supportedLocales: ['ru', 'en'], // Поддерживаемые языки
};
```

## 💾 Хранение

Язык сохраняется в:

1. **localStorage** — ключ `portfolio_locale`
2. **URL параметр** — `?lang=en`
3. **Язык браузера** — автоопределение при первом посещении

Приоритет:

1. Параметр в URL (`?lang=en`)
2. Сохранённый в localStorage
3. Язык браузера
4. Язык по умолчанию (ru)

## 🎨 UI Переключатель

Модуль `LocaleModule` автоматически создаёт переключатель в навигации:

```
[🌐 RU] ← Кнопка в навигации
```

При клике язык переключается между RU и EN.

### Стилизация

```css
.locale-toggle {
  /* Кнопка переключателя */
}

.locale-icon {
  /* Иконка глобуса */
}

.locale-text {
  /* Текст языка (RU/EN) */
}
```

## 🔄 Динамическое обновление

При переключении языка:

1. Обновляются все элементы с `data-i18n`
2. Обновляются `placeholder` у input с `data-i18n-placeholder`
3. Обновляется `lang` атрибут у `<html>`
4. Обновляется `title` и `meta description`
5. Сохраняется в localStorage
6. Обновляется URL параметр

## 🎯 Примеры

### Программное переключение

```typescript
// Переключить на английский
I18n.setLocale('en');

// Получить все переводы
const translations = I18n.getTranslations();

// Проверить поддержку
if (I18n.isSupported('en')) {
  I18n.setLocale('en');
}
```

### В компоненте

```typescript
import { I18n } from '@core/i18n';

class MyComponent {
  init() {
    // Подписка на изменения
    I18n.subscribe(() => this.render());
    this.render();
  }

  render() {
    const title = I18n.t('myComponent.title');
    // Обновить UI
  }
}
```

## 🐛 Отладка

```typescript
// В консоли браузера
I18n.getLocale(); // Текущий язык
I18n.getTranslations(); // Все переводы
I18n.t('nav.home'); // Перевод ключа
I18n.setLocale('en'); // Сменить язык
```
